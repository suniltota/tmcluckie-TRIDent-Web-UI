Escrow Item Label

A label to ensure clear and concise messaging of the escrow item

  • Predefined to support Homeowners Insurance, Mortgage Insurance and Property Taxes
  • These predefined are not editable

Rows below the predefined are editable fields for the user to add additional items

  • Format is a string value
 		Escrow Type

A list of enumerated values in which a user must choose to support the UCD xml file, the predefined lines will not get an escrow type as these are already set in the system

  • Format is a list of enumerated values
 		Amount Per Month

Each item must provide a value to indicate the monthly amount of the escrow item

  • Format is an amount value 9.2
 		Number of Months

Each item must provide a value for the number of months that will support the cushion of the escrow.

  • Format is numeric 3
 		Amount Include in APR?

Fee Label

A free-form text field that provides a human-readable, display-ready representation for the corresponding enumerated value.

  • Typically, the name a lender calls a specific fee, the user would type that fee in this location.
  • Format is a String Value up to 150 characters
 		Fee Type

Specify the general type of fees associated with a real estate finance transaction.

  • Required to support the GSE xml file, even though a Fee Label Name is provided, the user must choose an enumerated value to associate to the label name.
  • If other is chosen, then the user will be provided another text box (see below)
 		Amount Include in APR?
