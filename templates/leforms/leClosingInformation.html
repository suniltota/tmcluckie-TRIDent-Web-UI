Loan Estimate General Information
Lender Name Name of the creditor extending the estimated cost for obtaining a loan for real property.
Date Issued

The date the integrated disclosure was mailed or delivered to the borrower as reflected on a single instance of the integrated disclosure document.

  • Must be in a format of DD/MM/YYYY or use the calendar icon to choose the date.
  • There is no constraint on this date as a user may need to create a .xml file to submit to an investor to generate an integrated Loan Estimate document from the past.
  • Is a Required field
Closing Cost Expiration Date

The date in which the closing costs are expired and a new Loan Estimate must be issues, see CFPBs Integrate Mortgage Disclosure Regulation for more information on timing.

Street Address Legal Description
Street Address The address with the address number, pre-directional, street name, post-directional, address unit designators and address unit value
Legal Description The full, unparsed legal description of a parcel of real property.
  • The format is a Dollar amount in $9.2 Must be in a format of a String value with no more than 150 characters
City The name of the city.
  • Must be in a format of a String value with no more than 50 characters format is a string Value with a character limit of 50 characters.
State The two-character representation of the US state, US Territory, Canadian Province, Military APO FPO, or Territory.
Postal Code * The postal code (ZIP Code in the US) for the address. ZIP Code may be either 5 or 9 digits.
  • This value is required regardless if of other property Information is available.
