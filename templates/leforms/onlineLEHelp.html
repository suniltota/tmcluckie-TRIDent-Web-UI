General Information
Web User Interface - Label Name Online Help Verbiage
Lender Name
Name of the creditor extending the estimated cost for obtaining a loan for real property.
Date Issued
The date the integrated disclosure was mailed or delivered to the borrower as reflected on a single instance of the integrated disclosure document.
  • • Must be in a format of DD/MM/YYYY or use the calendar icon to choose the date.
  • • There is no constraint on this date as a user may need to create a .xml file to submit to an investor to generate an integrated closing disclosure document from the past
  • • Is a Required field
Closing Date
The date the closing documents were fully executed and the borrower becomes contractually responsible for the debt. Depending on the location (wet-sign, dry-sign, table-funded and/or escrow state) of the loan closing, the Closing Date may be the same date as either the Document Preparation Date and/or the Execution Date and/or the Recorded Date. • This date is defaulted to 5 days after the issue date if conducting data entry. Otherwise, the date will be rendered per the information in the upload process will be displayed • Must be in a format of DD/MM/YYYY or use the calendar icon to choose the date. • There is no constraint on this date as a user may need to create a .xml file to submit to an investor to generate an integrated closing disclosure document from the past. • Is a Required field
Disbursement Date

The date the loan funds are disbursed to the borrower or another party on the borrower's behalf.

  • • Must be in a format of DD/MM/YYYY or use the calendar icon to choose the date.
  • • Constraint on the date as it cannot be a date prior to the issue date or the closing date.
  • • Is a Required field
