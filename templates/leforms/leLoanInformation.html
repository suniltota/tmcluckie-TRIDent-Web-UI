Loan Information

Loan Period, Type and Position
Loan Term:
No. Years

In each transaction in accordance with the CFPB “Know Before You owe” Regulation must provide the entire term of the loan in years. Using the text box must input the number of years of the term

  • Example, lenders typically use months so 360 months equates to 30 years
  • Format is numeric up to 3 digits.
No. Months

In some transactions, such as construction to permanent or construction the term may be less than a year, so the user must input the number of months.

  • Example Construction Loan of 11 months
  • Example Construction to Permanent with a single close would have both the number of years and the number of months potentially, ex: 30 years and 11 months.
  • Format is numerical up to 2 digits
Loan Type

Identifies the highest level private or public sector entity under whose guidelines the mortgage is originated.

  • Acceptable values are: Conventional | FHA | Local Agency | Other | Public and Indian Housing | State Agency | USDA Rural Development | VA
  • Format is are the only enumerated values seen above.

Other Description

Purpose

Specifies the purpose for which the loan proceeds will be used. This field is not uneditable as it is set when the user logs in to the application.

  • If the transaction needs to change the user must start a new transaction, cannot change the purpose.
Lien Position

Specifies the priority of the lien against the subject property

  • Format is enumerated values that include the following: First Lien | Fourth Lien | Second Lien | Third Lien
Lender Loan Identifier

The unique loan identifier for a specific transaction for a specific lender

  • Format is a string value up to 30 characters.

This is a required value and must be present in the transaction

Product Description

A description of the loan product which includes, when applicable, certain loan features added to the loan product that may change the periodic loan payment

  • Required for every transaction

Home Equity Loan Information
Is the Loan a Home Equity Loan?

In all transaction, must indicate the loan is an extension of credit that does not involve the purchase of real property or the initial construction of a dwelling and will not be used to refinance an existing obligation.

In other words, this may be similar to a refinance transaction; however, in accordance with the CFPB “Know Bbefore you You oweOwe”, the different is there are NO liens against the subject property to pay off.

  • Required to be addressed

Boolean values of Yes or No in a drop down.
