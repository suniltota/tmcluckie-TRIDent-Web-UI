Property Information
Choose One

In any transaction, the subject property location must be made clear and concise on the Closing Disclosure, thus the user interface provides two options:

  • Street Address
  • Legal Description

The function requires that a radio button is chosen to indicate one or the other as this field is required.
Street Address

The address with the address number, pre-directional, street name, post-directional, address unit designators and address unit value

 Legal Description

The full, unparsed legal description of a parcel of real property.

  • Must be in a format of a String value with no more than 150 characters

Price Information (Purchase Only)
Personal Property Included?

In a purchase transaction, the regulation requires that if personal property is included and itemized separately in the sales contract that this must be disclosed. The user is provided a dropdown box to indicate if personal property is included or not.

  • The format is a Boolean choice of Yes or No
  • Is a required field
Sales Price

In a transaction where there is NO personal property itemized and included then provide the amount of money the contract is for.

  • The format is an amount that cannot exceed $9.2
  • Is considered conditionally required based of the personal property included indicator
$
Real Property Price

In a transaction where there IS personal property itemized and included then provide the dollar amount of real property included in the Sales Contract Amount.

  • The format is an amount that cannot exceed $9.2
  • Is considered conditionally required based of the personal property included indicator
$
Value Information
Choose either “Appraised Property Value” or “Estimated Property Value”

If the transaction is a Refinance, choose from the drop-down menu then select either the Appraised Property Value or the Estimated Property Value.

  • See the Know before you Owe regulation to determine which one to choose.
  • The Default is the Appraised Property Value
  • The format is a set of enumerated values seen in the drop-down box
Property Value Method Used

Specifies the method by which the property value was assessed.

  • The format is a set of enumerated values seen in the drop-down box, including “Other”
  • Is considered conditionally required if the user is submitting to either GSE and selected “Appraised Property Value”
If, Other is the property Value Method Used provide the description

A free-form text field used to capture a description of the Property Valuation Method Type when Other is selected.

  • The format is a string value that cannot exceed 150 characters
Amount of Value

Provide the Statement of value of the property from a valid property valuation source

  • This amount is different than the estimated property value amount
  • The format is a Dollar amount in $9.2

Provide the estimated present market value of the property that is from the borrower or loan originator.

  • This amount is different than the Appraised property value amount
  • The format is a Dollar amount in $9.2
