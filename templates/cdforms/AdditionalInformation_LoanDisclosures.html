Loan Disclosures

Assumption

A set of check boxes, as seen in the CD PDF, the user may choose one or the other to indicate if an assumption is allowed or not under the original terms

 		Will allow, under certain conditions, this person to assume this loan on the original terms Will Not allow assumption on this loan on the original terms
Demand Feature

A set of check boxes, as seen in the CF PDF, the user may choose one or the other to identify if the transaction has a demand feature, which permits your lender to require early repayment of the loan or not.

 		Has a demand feature, which permits your lender to require early repayment of the loan. Does not have a demand feature
Late Charges

Button: Input State Values

A user may click on the button called Input State Values which will fill in the appropriate information to support the number of days grace period, any amount or percentage and the type if a late charge occurs.

Or the user may enter in the values for the following text boxes

  • Number of Grace Period Days
  • Late Fee if a Percent
  • Late Fee if an amount
  • Late Charge Type – describes of what the fee is a percent of or an amount of…

Note that in the CFPB Integrated Mortgage Disclosure Regulation there is guidance as to when to display percentage vs Amount or both please validate against the regulation to provide input guidance.

 		Number of Grace Period Days Late Fee if a Percent
%
Late Fee if an Amount
$
Late Charge Type
Partial Payment

User may choose one to three of the check boxes regarding the situation of the lender for partial payments, please see the CFPBs Integrated Mortgage Disclosure Regulation for guidance.

 		May Accept Payments that are less than full amount due and apply to the loan May hold them in a separate account until a full payment is received Does not accept any partial payment
Negative Amortization

The values to support the CD PDF are found in the Product Description section in the Loan Information, there is a link in the Disclosure section that is called “Product Description” that takes the user to the information.

 		See the Product Description Section for information on Negative Amortization
